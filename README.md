## 🇬🇧 English

### 🎯 Overview
*React CRUD App is a modular single-page application developed by Erfan Ahmadi, built with React to practice fundamental CRUD operations — Create, Read, Update, and Delete. The project demonstrates strong understanding of React routing, reusable components, and state management. It was built as part of a personal learning journey to deepen practical experience with modern front-end development.

### 🧩 Features
- Modular component-based structure.
- CRUD functionality (Create, Read, Update, Delete) for managing products.
- Built using React and Vite (or similar setup).
- Routing handled with React Router.
- Clean folder architecture and scoped CSS per component.
- Responsive design principles applied.

### ⚙️ Installation & Setup
# Clone the repository
git clone https://github.com/er4Nxz/React-crud.git
cd React-crud

# Install dependencies
npm install

# Run the development server
npm run dev

### 🛠️ Technologies Used
- React (Hooks, Components, State)
- React Router DOM (Routing)
- Axios (HTTP client)
- MockAPI (remote REST data source)
- SweetAlert2 (modals/alerts)
- React Icons
- Swiper (home page slider)
- React Loading Skeleton (skeleton states)
- CSS Modules / Custom Styles
- JavaScript (ES6)
